# 📱 Cartão Digital de Identificação

Aplicação desenvolvida com **Ionic + Angular** que simula um cartão digital de identificação de estudante, com interface moderna, responsiva e navegação fluida.

---

## 🚀 Sobre o Projeto

Este projeto tem como objetivo apresentar um **cartão digital interativo**, exibindo informações acadêmicas de forma visual e organizada.  

A aplicação foi construída utilizando **Angular** com **Ionic Framework**, proporcionando uma experiência semelhante a aplicativos mobile.

---

## ✨ Funcionalidades

- 📇 Exibição de cartão digital com dados do estudante  
- 👤 Página "Sobre" com informações pessoais e acadêmicas  
- 🔄 Navegação entre páginas  
- 📱 Design responsivo (mobile e desktop)  
- 🎨 Interface moderna com gradientes e animações  

---

## 🛠️ Tecnologias Utilizadas

- **Angular 20+**
- **Ionic 8**
- **TypeScript**
- **SCSS**
- **Capacitor**
- **RxJS**
